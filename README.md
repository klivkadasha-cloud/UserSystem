# User Management System (Console App)

## 📌 Description
This is a simple console-based Java application for managing users.  
The project demonstrates basic Object-Oriented Programming (OOP) concepts such as inheritance and polymorphism.

---

## 🚀 Features
- Add users
- Support different user types (Admin, RegularUser)
- Search user by name
- Display all users
- Use of inheritance and method overriding

---

## 🧠 OOP Concepts Used
- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism
- Method overriding

---

## 🏗️ Project Structure
- `User` – base class for all users
- `Admin` – extended user with additional role
- `RegularUser` – standard user type
- `Main` – application entry point

---

## ▶️ How to Run
1. Open project in IntelliJ IDEA (or any Java IDE)
2. Run `Main.java`
3. Follow console instructions

---

## 💡 Example Flow
- User selects type (Admin / RegularUser)
- Enters name and age
- System stores user in memory
- User can search by name
- System displays user information

---

## 📚 Purpose
This project was created for learning Java OOP fundamentals and practicing working with collections and user input.
